# AI Game Lab — 10 Games

All games run directly in the browser. No install, no server, no dependencies.
Just open the index.html inside each folder.

## Games

| # | Folder           | Algorithm | Description                                      |
|---|------------------|-----------|--------------------------------------------------|
| 1 | flappy-bird      | NEAT      | 50 birds evolve to thread pipes                  |
| 2 | snake            | DQN       | Agent learns to eat food, Q-value heatmap        |
| 3 | car-racing       | NEAT      | 30 cars drive a procedural oval track            |
| 4 | pong             | DQN       | Two agents in self-play                          |
| 5 | asteroid-dodge   | NEAT      | 30 ships dodge infinite asteroids                |
| 6 | game-2048        | MCTS      | Monte Carlo Tree Search plays 2048               |
| 7 | breakout         | DQN       | Agent breaks bricks, epsilon decay shown live    |
| 8 | predator-prey    | GA+NEAT   | 100 prey + 20 predators evolve independently     |
| 9 | lunar-lander     | NEAT      | 20 landers learn to touch down safely            |
|10 | maze-solver      | GA        | 200 walkers + pheromone trails + path replay     |

## YouTube streaming tip
1. Open any game fullscreen in Chrome (F11)
2. OBS → Add Source → Window Capture → select Chrome
3. Stream to YouTube Live via RTMP
