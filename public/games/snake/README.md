# Snake — Hamiltonian Cycle + A* Shortcuts
A planning AI (not learning). It follows a precomputed Hamiltonian cycle for guaranteed
survival, taking safe A*/greedy shortcuts straight to the apple whenever a move is proven
safe (it can still reach its own tail). Clears the full board (score 397) every game.
The side panel shows the real cycle, the live planned path, and live stats.
Open index.html in any browser. No install needed.
