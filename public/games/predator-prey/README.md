# Predator Prey — Co-evolving Neural Brains (GA)
40 prey and 4 predators each evolve their own neural networks with a genetic algorithm.
Prey see the 2 nearest predators + nearest plant (11 inputs); predators see the 2 nearest
prey (9 inputs). A "red queen" arms race drives emergent hunting and fleeing. Population
chart and live network on the right.
Open index.html in any browser. No install needed.
